# v0.2.0: Testing & CI/CD — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add GitHub Actions CI/CD pipelines and expand unit test coverage from 18.7% to 35-40% via httptest-based API layer tests.

**Architecture:** Two GitHub Actions workflows (CI on push/PR, Release on tag). Then systematically add httptest-based tests for each API service (9 test files, 100+ methods). A shared `testutil_test.go` provides `newTestClient()` helper. Each API test uses `CONOHA_ENDPOINT` env var to redirect `baseURL()` to httptest server.

**Tech Stack:** Go 1.26, `net/http/httptest`, GitHub Actions, goreleaser.

**Spec:** `docs/superpowers/specs/2026-03-28-v020-testing-ci-design.md`

**User preferences:**
- Git commit messages in English, no Co-Authored-By
- Do NOT test `CONOHA_ENDPOINT_MODE` branches (internal endpoint info)
- No internal hostnames in code or tests

- [ ] **Step 2: Write compute API tests**

`internal/api/compute_test.go` — test ALL 19 public methods of ComputeAPI. Each test:
1. Creates an httptest.Server with a handler that verifies the HTTP method, URL path, and (for POST/PUT) request body JSON structure
2. Returns appropriate JSON response
3. Uses `t.Setenv("CONOHA_ENDPOINT", ts.URL)` to redirect baseURL
4. Calls the API method and asserts the response

Methods to test (grouped by HTTP pattern):

**GET methods (List/Get):** `ListServers`, `GetServer`, `FindServer` (UUID path), `ListFlavors`, `GetFlavor`, `ListKeypairs`, `ListVolumeAttachments`, `GetServerMetadata`

**POST methods (Create):** `CreateServer`, `CreateKeypair`, `GetConsole`

**POST action methods:** `StartServer`, `StopServer`, `RebootServer`, `ResizeServer`, `RebuildServer`
- These call `ServerAction()` internally. Test via the public wrapper (e.g., `StartServer` sends `{"os-start": nil}`).
- Note: `ServerAction` and `AttachVolume` use `Client.Request()` directly and close body manually. The handler should return status 202 with empty body.

**PUT methods:** `RenameServer`

**DELETE methods:** `DeleteServer`, `DeleteKeypair`, `DetachVolume`

**POST with raw response:** `AttachVolume` — handler returns 200, body is closed by caller.

Example for a representative subset:

```go
func TestListServers(t *testing.T) {
	ts := httptest.NewServer(http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
		if r.Method != "GET" {
			t.Errorf("expected GET, got %s", r.Method)
		}
		if r.URL.Path != "/v2.1/servers/detail" {
			t.Errorf("unexpected path: %s", r.URL.Path)
		}
		w.Header().Set("Content-Type", "application/json")
		fmt.Fprint(w, `{"servers":[{"id":"srv-1","name":"test-server","status":"ACTIVE"}]}`)
	}))
	defer ts.Close()

	t.Setenv("CONOHA_ENDPOINT", ts.URL)
	api := NewComputeAPI(newTestClient(ts))
	servers, err := api.ListServers()
	if err != nil {
		t.Fatal(err)
	}
	if len(servers) != 1 {
		t.Fatalf("expected 1 server, got %d", len(servers))
	}
	if servers[0].ID != "srv-1" {
		t.Errorf("expected id srv-1, got %s", servers[0].ID)
	}
}

func TestCreateServer(t *testing.T) {
	ts := httptest.NewServer(http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
		if r.Method != "POST" {
			t.Errorf("expected POST, got %s", r.Method)
		}
		if r.URL.Path != "/v2.1/servers" {
			t.Errorf("unexpected path: %s", r.URL.Path)
		}
		var body map[string]any
		json.NewDecoder(r.Body).Decode(&body)
		server := body["server"].(map[string]any)
		if server["name"] != "my-server" {
			t.Errorf("unexpected name: %v", server["name"])
		}
		w.Header().Set("Content-Type", "application/json")
		w.WriteHeader(201)
		fmt.Fprint(w, `{"server":{"id":"new-id","name":"my-server","status":"BUILD"}}`)
	}))
	defer ts.Close()

	t.Setenv("CONOHA_ENDPOINT", ts.URL)
	api := NewComputeAPI(newTestClient(ts))
	req := &model.ServerCreateRequest{Server: model.ServerCreate{Name: "my-server", FlavorRef: "flavor-1", ImageRef: "image-1"}}
	server, err := api.CreateServer(req)
	if err != nil {
		t.Fatal(err)
	}
	if server.ID != "new-id" {
		t.Errorf("expected new-id, got %s", server.ID)
	}
}

func TestStartServer(t *testing.T) {
	ts := httptest.NewServer(http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
		if r.Method != "POST" {
			t.Errorf("expected POST, got %s", r.Method)
		}
		if !strings.HasSuffix(r.URL.Path, "/action") {
			t.Errorf("expected action path, got %s", r.URL.Path)
		}
		var body map[string]any
		json.NewDecoder(r.Body).Decode(&body)
		if _, ok := body["os-start"]; !ok {
			t.Error("expected os-start action")
		}
		w.WriteHeader(202)
	}))
	defer ts.Close()

	t.Setenv("CONOHA_ENDPOINT", ts.URL)
	api := NewComputeAPI(newTestClient(ts))
	if err := api.StartServer("srv-1"); err != nil {
		t.Fatal(err)
	}
}

func TestDeleteServer(t *testing.T) {
	ts := httptest.NewServer(http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
		if r.Method != "DELETE" {
			t.Errorf("expected DELETE, got %s", r.Method)
		}
		w.WriteHeader(204)
	}))
	defer ts.Close()

	t.Setenv("CONOHA_ENDPOINT", ts.URL)
	api := NewComputeAPI(newTestClient(ts))
	if err := api.DeleteServer("srv-1"); err != nil {
		t.Fatal(err)
	}
}
```

Follow this same pattern for ALL 19 methods. Use `t.Run()` subtests for related methods (e.g., all action methods under `TestServerActions`).

**Important notes for specific methods:**
- **`FindServer`**: Has 3 code paths (UUID lookup, name search via ListServers, fallback ID). The handler must serve both `/servers/detail` and `/servers/{id}` paths. Use a mux or path-switching handler.
- **`ListKeypairs`**: Response has nested unwrapping: `{"keypairs": [{"keypair": {...}}]}` (not flat `{"keypairs": [{...}]}`). Check `model.KeypairsResponse` and `model.KeypairWrapper` types.
- **`ServerAction`/`AttachVolume`**: Use `Client.Request()` directly and close body manually. Handler should return 202 with empty body.
- **Test handlers should NOT return 5xx accidentally** — `Client.Do()` retries on 5xx with sleep, causing slow tests.

func TestAuthenticate(t *testing.T) {
	ts := httptest.NewServer(http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
		if r.Method != "POST" || r.URL.Path != "/v3/auth/tokens" {
			t.Errorf("unexpected: %s %s", r.Method, r.URL.Path)
		}
		w.Header().Set("X-Subject-Token", "new-token-123")
		w.Header().Set("Content-Type", "application/json")
		w.WriteHeader(http.StatusCreated) // auth.go checks for 201, not 200
		fmt.Fprint(w, `{"token":{"expires_at":"2099-12-31T23:59:59Z"}}`)
	}))
	defer ts.Close()

	t.Setenv("CONOHA_ENDPOINT", ts.URL)
	result, err := Authenticate("c3j1", "tenant-id", "user", "pass")
	if err != nil {
		t.Fatal(err)
	}
	if result.Token != "new-token-123" {
		t.Errorf("expected new-token-123, got %s", result.Token)
	}
}

func TestEnsureTokenFromEnv(t *testing.T) {
	t.Setenv("CONOHA_TOKEN", "env-token")
	token, err := EnsureToken("default", nil, nil, nil)
	if err != nil {
		t.Fatal(err)
	}
	if token != "env-token" {
		t.Errorf("expected env-token, got %s", token)
	}
}

func TestEnsureTokenCached(t *testing.T) {
	// Pre-populate token store with a valid (future-expiring) token
	dir := t.TempDir()
	tokens := config.NewTokenStore(dir + "/tokens.yaml")
	tokens.Set("default", "cached-tok", "2099-12-31T23:59:59Z")

	t.Setenv("CONOHA_TOKEN", "") // clear env override
	token, err := EnsureToken("default", nil, nil, tokens)
	if err != nil {
		t.Fatal(err)
	}
	if token != "cached-tok" {
		t.Errorf("expected cached-tok, got %s", token)
	}
}
